# Цены (Ozon FBO)

## `POST /v1/product/import/prices`

Обновление цен товаров. До 1000 товаров за запрос.

### Параметры (элемент массива `prices`)
| Параметр | Тип | Описание |
|----------|-----|----------|
| `offer_id` | string | Артикул |
| `product_id` | int | ID товара |
| `price` | string | Цена с учётом скидок |
| `old_price` | string | Цена до скидки (0 – сбросить) |
| `min_price` | string | Минимальная цена |
| `vat` | string | НДС |
| `currency_code` | string | Валюта (по умолчанию RUB) |

### Особенности
- Цену каждого товара можно обновлять не чаще 10 раз в час.

## `POST /v5/product/info/prices`

Получение информации о ценах, комиссиях.

### Параметры
| Параметр | Тип | Описание |
|----------|-----|----------|
| `filter.offer_id` | array | Артикулы |
| `filter.product_id` | array | ID товаров |
| `limit` | int | До 1000 |
| `cursor` | string | Для пагинации |

### Поля ответа
| Поле | Описание |
|------|----------|
| `items[].price.price` | Текущая цена |
| `items[].price.old_price` | Цена до скидки |
| `items[].price.min_price` | Минимальная цена |
| `items[].commissions` | Комиссии по схемам |

## `POST /v1/product/update/discount`

Установка скидки на уценённый товар (FBS).

### Параметры
| Параметр | Тип | Описание |
|----------|-----|----------|
| `product_id` | int | ID товара |
| `discount` | int | Размер скидки (3-99%) |
