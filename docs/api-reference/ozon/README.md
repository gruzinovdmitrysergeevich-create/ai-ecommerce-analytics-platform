# Ozon Seller API – структурированная документация (FBO)

Документация разделена на две категории: **Аналитика и отчёты** и **Автоматизация**. Это позволяет быстро находить методы под конкретную задачу.

## 🔹 Аналитика и отчёты

Методы для получения финансовых, статистических и операционных данных.

| Раздел | Файл | Описание |
|--------|------|----------|
| Финансовые отчёты | [finance.md](analytics/finance.md) | Транзакции, реализация, компенсации |
| Основная аналитика | [analytics_data.md](analytics/analytics_data.md) | `/v1/analytics/data` – продажи, показы, корзина |
| Остатки и оборачиваемость | [stocks.md](analytics/stocks.md) | Аналитика остатков на складах FBO |
| Отправления FBO | [postings.md](analytics/postings.md) | Списки и отчёты по отправлениям |
| Возвраты | [returns.md](analytics/returns.md) | Информация о возвратах товаров |
| Рейтинги | [ratings.md](analytics/ratings.md) | Рейтинги продавца и индексы |

## 🔸 Автоматизация

Методы для управления товарами, ценами, заказами и складскими операциями.

| Раздел | Файл | Описание |
|--------|------|----------|
| Товары и атрибуты | [products.md](automation/products.md) | Загрузка, обновление, категории |
| Цены | [prices.md](automation/prices.md) | Управление ценами и скидками |
| Остатки (FBO – чтение) | [stocks_management.md](automation/stocks_management.md) | Получение информации об остатках |
| Заказы и поставки FBO | [orders_fbo.md](automation/orders_fbo.md) | Управление поставками, пропуска |
| Склады и кластеры | [warehouses.md](automation/warehouses.md) | Списки складов, кластеры |
| Акции и стратегии | [actions.md](automation/actions.md) | Участие в акциях, ценообразование |
| Сертификаты | [certificates.md](automation/certificates.md) | Загрузка сертификатов на товары |
| Чаты | [chats.md](automation/chats.md) | Работа с чатами покупателей |
| Пуш-уведомления | [notifications.md](automation/notifications.md) | Настройка уведомлений о событиях |
| **Реклама** | [ads.md](automation/ads.md) | Performance API: статистика кампаний, атрибуция |

## 📌 Примечание

- Все методы актуальны для схемы **FBO** (продажа со склада Ozon).
- Для каждого метода указаны эндпоинт, назначение, ключевые параметры и ссылка на официальную документацию.
