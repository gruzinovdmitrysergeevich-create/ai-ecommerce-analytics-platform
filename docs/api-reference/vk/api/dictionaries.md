
Справочники
Статические данные для настройки таргетингов.

Регионы
GET /api/v2/regions.json – все регионы (фильтры _parent_id, _q, _flags).

Пользовательская география
GET /api/v2/user_geo.json – регионы из профилей соцсетей.

Мобильные справочники
Категории приложений: GET /api/v2/mobile_categories.json

ОС: GET /api/v2/mobile_os.json

Операторы: GET /api/v2/mobile_operators.json

Типы устройств: GET /api/v2/mobile_types.json

Производители: GET /api/v2/mobile_vendors.json

Интересы
GET /api/v2/targetings_tree.json – полное дерево интересов.

Поля и паттерны баннеров
GET /api/v2/banner_fields.json

GET /api/v2/banner_patterns.json
