# Builder public

## GET /api/builder/domains/published/by_id/{builder_id}/

****  
*operationId: `get_public_builder_by_id`*

**Параметры:**

- `builder_id` (path) — Returns the builder related to the provided Id and its pages.

---

## GET /api/builder/domains/published/by_name/{domain_name}/

****  
*operationId: `get_public_builder_by_domain_name`*

**Параметры:**

- `domain_name` (path) — Returns the builder published for the given domain name.

---

## GET /api/custom_code/{builder_id}/css/

****  
*operationId: `get_public_builder_custom_code`*

**Параметры:**

- `builder_id` (path) — The builder Id we want the custom code for.
- `type` (path) — Type of code to return: 'css' or 'js'
