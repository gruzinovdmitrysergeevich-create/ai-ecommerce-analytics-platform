# n8n REST API Documentation

Документация по API n8n для self-hosted инстанса (Docker, порт 5678).

> **Обязательно прочитайте [general.md](general.md)** — там базовая информация по аутентификации, базовому URL и пагинации.

## Содержание
- [general.md](general.md) – Аутентификация, базовый URL, пагинация, особенности self-hosted
- [workflows.md](workflows.md) – Управление рабочими процессами
- [executions.md](executions.md) – Управление выполнениями
- [credentials.md](credentials.md) – Учётные данные для внешних сервисов
- [variables.md](variables.md) – Глобальные и проектные переменные
- [projects.md](projects.md) – Проекты и участники
- [tags.md](tags.md) – Теги для организации
- [users.md](users.md) – Пользователи инстанса (только для owner)
- [audit.md](audit.md) – Генерация отчёта по безопасности
- [source_control.md](source_control.md) – Работа с Git
- [data_tables.md](data_tables.md) – Таблицы данных (Data Tables API)
- [discover.md](discover.md) – Доступные эндпоинты

Официальная документация: [https://docs.n8n.io/api/](https://docs.n8n.io/api/)
