# Поставки FBS

## `POST /api/v3/supplies`

**Назначение:** создать новую поставку.  
**Тело:** `{"name": "Поставка 01.01"}`.  
**Ответ:** `{"id": "WB-GI-..."}`.

## `PATCH /api/marketplace/v3/supplies/{supplyId}/orders`

**Назначение:** добавить сборочные задания к поставке (до 100).  
**Тело:** `{"orders": [123, 456]}`.  
**Эффект:** задания переходят в статус `confirm`.

## `PATCH /api/v3/supplies/{supplyId}/deliver`

**Назначение:** передать поставку в доставку.  
**Условия:** в поставке есть задания, нет ошибок маркировки.

## `GET /api/v3/supplies/{supplyId}/barcode`

**Назначение:** получить QR-код поставки (после передачи в доставку).
