# Реклама Wildberries (API)

Базовый URL: `https://advert-api.wildberries.ru`

## Авторизация
Заголовок `Authorization: Bearer <токен>`

## `POST /adv/v1/upd`
Статистика по рекламным кампаниям (детализация по дням, кампаниям, баннерам).

### Параметры
- `campaignIds` – массив ID кампаний
- `dateFrom`, `dateTo` – период (YYYY-MM-DD)
- `groupBy` – `campaign`, `banner`, `day`

### Пагинация
Нет, возвращает всё за период.

## `POST /adv/v3/fullstats`
Агрегированная статистика (суммарные показатели).

### Параметры
Аналогично `/upd`.

## Особенности
- Рекламный кабинет использует отдельный токен (не тот же, что для статистики продаж).
- Данные за последние 90 дней.

[Официальная документация](https://dev.wildberries.ru/openapi/advert-api)
