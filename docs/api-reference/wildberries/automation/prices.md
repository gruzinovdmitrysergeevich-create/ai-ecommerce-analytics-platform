# Цены и скидки (FBS)

## `POST /api/v2/upload/task`

**Назначение:** установить цены и скидки для товаров.  
**Тело:** `{"data": [{"nmID": 123, "price": 999, "discount": 30}]}` (до 1000).  
**Ответ:** `{"data": {"id": 12345, "alreadyExists": false}}`.

**Особенности:**  
- Цена и скидка не могут быть пустыми одновременно.  
- Если новая цена со скидкой в 3 раза меньше старой, товар попадёт в карантин.

---

## `GET /api/v2/history/tasks`

**Назначение:** статус загрузки цен.  
**Параметры:** `uploadID`.  
**Ответ:** `status` (`3` — обработана, `5` — с ошибками).

---

## `GET /api/v2/list/goods/filter`

**Назначение:** получить текущие цены и скидки товаров.  
**Параметры:** `limit`, `offset`, `filterNmID`.
