# Управление остатками (FBS)

## `PUT /api/v3/stocks/{warehouseId}`

**Назначение:** обновить остатки товаров.  
**Тело:** `{"stocks": [{"chrtId": 123, "amount": 10}]}` (до 1000).  

## `DELETE /api/v3/stocks/{warehouseId}`

**Назначение:** удалить запись об остатках.  
**Тело:** `{"chrtIds": [123, 456]}`.
