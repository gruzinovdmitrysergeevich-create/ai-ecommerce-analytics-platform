# Общая информация о Mobzio API

## Базовый URL
https://mobz.io/api/public/

text

## Авторизация
API ключ необходимо получить в поддержке Mobzio.  
Ключ передаётся в заголовке каждого запроса:
Authorization: ваш_api_ключ

text

## Формат запросов и ответов
- Методы: `GET`, `POST`.
- Данные в теле запроса для `POST` обычно передаются как `application/x-www-form-urlencoded` или `multipart/form-data` (если требуется загрузка файлов).
- Все ответы возвращаются в формате JSON.

### Стандартная структура ответа
```json
{
  "status": "success" или "error",
  "result": ...,
  "message": "..."
}
status – "success" или "error".

result – данные ответа (может быть массивом или объектом).

message – дополнительное сообщение (при ошибке – описание, при успехе может содержать полную ссылку).

Обработка ошибок
При ошибке статус "error", а поле message содержит описание проблемы.

Типы данных в параметрах
shortcode – короткий идентификатор ссылки (например, api3).

link_id – числовой идентификатор ссылки.

folder_id – числовой идентификатор папки.

dateFrom, dateTo – timestamp (Unix-секунды).

stats – флаг 1 для запроса статистики в ответе.

clean – флаг 1 для получения только уникальных кликов.
