# Mobzio API – документация для интеграции

Mobzio — сервис сокращения ссылок с расширенной аналитикой и интеграцией с маркетплейсами (Wildberries, Ozon и др.).

## ⚠️ Перед началом работы
**Обязательно прочитайте [`general.md`](general.md)** – базовый URL, авторизация, форматы ответов и особенности работы с API.

## Структура
- **[`general.md`](general.md)** – общие принципы API.
- **[`api/`](api/)** – методы, сгруппированные по темам:
  - [`links.md`](api/links.md) – создание, редактирование и получение ссылок.
  - [`statistics.md`](api/statistics.md) – статистика переходов, конверсии, WB‑данные.
  - [`folders.md`](api/folders.md) – управление папками для организации ссылок.

## Официальные источники
- https://mobz.io/api/public/ (документация в формате Playground)
