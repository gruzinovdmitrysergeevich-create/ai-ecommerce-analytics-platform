# Инструкция для AI-агента

## Назначение

AI-аналитика: авто-обнаружение таблиц, расчёт метрик, генерация выводов.

## Запуск

```bash
cd analytics && python3 analyst.py "вопрос"
```

## Структура

```
analytics/
├── analyst.py                # оркестратор
├── config.yaml               # настройки
└── src/
    ├── data_discovery.py     # авто-обнаружение таблиц
    ├── data_linker.py        # поиск связей
    ├── metric_engine.py      # расчёт метрик
    ├── model_provider.py     # AI-модели
    └── sandbox.py            # песочница
```

## Важные правила

- Не изменяй основной код без явного запроса.
- При ошибках сначала читай логи, затем диагностируй проблему.
- Не удаляй конфигурационные файлы без подтверждения.
- Ключи и пароли — только из переменных окружения.
