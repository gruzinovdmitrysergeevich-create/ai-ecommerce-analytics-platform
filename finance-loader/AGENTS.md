# Инструкция для AI-агента

## Назначение

Загрузка финансовых данных из Google Sheets, Excel и банковских выписок в Baserow.

## Запуск

```bash
cd finance-loader && python3 src/finance_loader.py
```

## Структура

```
finance-loader/
├── src/finance_loader.py     # основной скрипт
└── configs/                   # JSON-конфиги источников
```

## Важные правила

- Не изменяй основной код без явного запроса.
- При ошибках сначала читай логи, затем диагностируй проблему.
- Не удаляй конфигурационные файлы без подтверждения.
- Ключи и пароли — только из переменных окружения.
