# Product Context

## Для чего
Создать MCP-сервер (Model Context Protocol) на Python, который даёт агентам (Roo Code, Kilo Code) полный контроль над self-hosted Baserow через REST API. Это позволит агентам автоматически создавать workspace, базы данных, таблицы, поля и управлять данными без ручного вмешательства.

## Пользователи
- **Roo Code / Kilo Code** – подключают MCP-сервер и используют инструменты для работы с Baserow.
- **Разработчики** – используют сервер для автоматизации onboarding клиентов и создания структур данных.

## Ценности
- Полная автоматизация работы с Baserow через MCP.
- Агенты могут создавать и управлять структурами данных без ручного API.
- Единый стандартный интерфейс для всех операций с Baserow.
- Логирование всех действий для аудита и отладки.
