## Глобальные правила
- Хранятся в `~/.roo/rules/`.
- Изменить файл: `tee ~/.roo/rules/XX-name.md << '...'` или `nano`.

## OpenCode (CLI)
- Конфиг: `~/.config/opencode/opencode.json`, секция `instructions` (полные пути к файлам правил).
- Плагин `opencode-rules` (установлен) читает глобальные правила из `~/.config/opencode/rules` (симлинк на `~/.roo/rules`).
- Для локальных правил проекта создаётся симлинк `.opencode/rules` → `.roo/rules` (выполняется при создании проекта).
- Проверка: `opencode --help`, `opencode plugin list`.

## Общие напоминания
- Всегда проверяй актуальность правил в `~/.roo/rules/` и `.roo/rules/` проекта.
- Для работы с Baserow используй API, а не прямой SQL.
- При генерации кода – обработка ошибок и логирование.
