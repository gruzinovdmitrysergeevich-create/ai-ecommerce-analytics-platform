# Инструкция для AI-агента

## Назначение

Поддержка и диагностика загрузчика данных из API WB/Ozon в Baserow.

## Запуск

```bash
cd universal-api-loader && python3 src/universal_loader.py
```

## Структура

```
universal-api-loader/
├── src/universal_loader.py    # основной скрипт
├── src/validate_config.py     # валидатор JSON-конфигов
├── configs/ozon/              # конфиги отчётов Ozon
├── configs/wildberries/       # конфиги отчётов WB
└── docs/                      # документация
```

## Важные правила

- Не изменяй основной код без явного запроса.
- При ошибках сначала читай логи, затем диагностируй проблему.
- Не удаляй конфигурационные файлы без подтверждения.
- Ключи и пароли — только из переменных окружения.
