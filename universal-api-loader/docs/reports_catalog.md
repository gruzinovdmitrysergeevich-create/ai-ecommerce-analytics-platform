# 📚 Каталог отчётов и таблиц в Baserow

## Wildberries (FBS)

### Таблица `sales` – Продажи по дням
**Назначение:** ежедневные отчёты о проданных товарах, ценах, скидках, комиссиях и суммах к выплате.

**Поля:**
| Английское имя | Русское описание | Что означает |
|----------------|------------------|--------------|
| `date` | Дата продажи | День, за который сформирован отчёт |
| `lastChangeDate` | Дата последнего изменения | Когда запись обновлялась в системе WB |
| `warehouseName` | Склад | Название склада отгрузки |
| `warehouseType` | Тип склада | Например, "Склад продавца" или "Склад WB" |
| `countryName` | Страна | Страна продажи |
| `oblastOkrugName` | Область/округ | Регион доставки |
| `regionName` | Регион | Более детальный регион |
| `supplierArticle` | Артикул продавца | Ваш внутренний артикул товара |
| `nmId` | Номенклатурный номер WB | Уникальный идентификатор карточки товара на Wildberries |
| `barcode` | Штрихкод | Штрихкод товара |
| `category` | Категория | Категория товара по классификации WB |
| `subject` | Предмет | Тип товара (например, "Крем для лица") |
| `brand` | Бренд | Бренд товара |
| `techSize` | Размер | Технический размер |
| `incomeID` | ID поставки | Идентификатор поставки, в рамках которой был продан товар |
| `isSupply` | Признак поставки | true, если товар был в поставке |
| `isRealization` | Признак реализации | true, если товар продан |
| `totalPrice` | Полная цена | Цена без скидок |
| `discountPercent` | Процент скидки | Скидка продавца |
| `spp` | Скидка постоянного покупателя | Процент скидки WB для постоянных клиентов |
| `paymentSaleAmount` | Сумма продажи с учётом скидки WB | Сколько заплатил покупатель |
| `forPay` | К выплате продавцу | Сумма, которую WB перечислит вам после всех удержаний |
| `finishedPrice` | Финальная цена | Цена после всех скидок (включая СПП) |
| `priceWithDisc` | Цена со скидкой продавца | Цена после вашей скидки, но до СПП |
| `saleID` | ID продажи | Уникальный идентификатор факта продажи |
| `sticker` | Стикер | Идентификатор стикера (если применимо) |
| `gNumber` | Номер заказа | Номер заказа покупателя |
| `srid` | Уникальный идентификатор строки отчёта | Используется для дедупликации |

### Таблица `realization_detail` – Детализация реализаций (финансовый отчёт)
**Назначение:** подробный финансовый разбор каждой проданной единицы: комиссии, логистика, хранение, штрафы, рекламные бонусы и т.д. Основа для расчёта чистой прибыли.

**Основные поля (всего 88):**
| Английское имя | Русское описание | Что означает |
|----------------|------------------|--------------|
| `rrdId` | ID строки детализации | Уникальный идентификатор записи в отчёте реализации |
| `giId` | ID поставки | Идентификатор поставки |
| `nmId` | Номенклатура WB | Артикул WB |
| `vendorCode` | Артикул продавца | Ваш артикул |
| `title` | Название товара | |
| `quantity` | Количество | Количество проданных единиц |
| `retailPrice` | Розничная цена | Цена без учёта скидок |
| `retailAmount` | Розничная сумма | `retailPrice * quantity` |
| `salePercent` | Процент скидки продавца | |
| `commissionPercent` | Процент комиссии WB | Базовая комиссия площадки |
| `forPay` | К выплате | Итоговая сумма к перечислению за данную позицию |
| `deliveryAmount` | Стоимость доставки | |
| `returnAmount` | Стоимость возврата | |
| `deliveryService` | Услуга доставки | Плата за логистику |
| `paidStorage` | Платное хранение | |
| `penalty` | Штрафы | |
| `additionalPayment` | Доплата | |
| `spp` | СПП | Скидка постоянного покупателя |
| `ppvzSalesCommission` | Комиссия ПВЗ | |
| `acquiringFee` | Эквайринг | |
| `srid` | Уникальный ключ | Для дедупликации |

*(полный список полей см. в конфиге `realization_detail.json`)*

| Английское имя | Русское описание | Что означает |

| `product.nmId` | Номенклатура WB | |

| `product.vendorCode` | Артикул продавца | |
| `statistic.selected.openCount` | Просмотры карточки | |
| `statistic.selected.cartCount` | Добавлено в корзину | |
| `statistic.selected.orderCount` | Заказано | |
| `statistic.selected.buyoutCount` | Выкуплено | |
| `statistic.selected.buyoutSum` | Сумма выкупов | |
| `statistic.selected.cancelCount` | Отменено | |
| `statistic.selected.avgPrice` | Средняя цена | |
| `statistic.selected.conversions.addToCartPercent` | Конверсия в корзину, % | |
| `statistic.selected.conversions.cartToOrderPercent` | Конверсия корзина→заказ, % | |
| `statistic.selected.conversions.buyoutPercent` | Процент выкупа, % | |

## Ozon (FBO)

### Таблица `transactions` – Финансовые транзакции
**Назначение:** все движения средств: начисления за продажи, комиссии, доставка, возвраты.

| Английское имя | Русское описание | Что означает |
|----------------|------------------|--------------|
| `operation_id` | ID операции | Уникальный идентификатор транзакции |
| `operation_type` | Тип операции | Код типа (например, `OperationType`) |
| `operation_type_name` | Название типа | Человеко-читаемое описание |
| `operation_date` | Дата операции | |
| `amount` | Сумма | Сумма операции (может быть отрицательной) |
| `type` | Направление | `income` / `expense` |
| `accruals_for_sale` | Начисления за продажу | |
| `sale_commission` | Комиссия за продажу | |
| `delivery_charge` | Стоимость доставки | |
| `return_delivery_charge` | Стоимость обратной доставки | |
| `posting.posting_number` | Номер отправления | |
| `posting.order_date` | Дата заказа | |
| `posting.delivery_schema` | Схема доставки | |
| `posting.warehouse_id` | ID склада | |
