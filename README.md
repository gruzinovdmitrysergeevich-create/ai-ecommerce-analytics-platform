# 🧠 AI-Powered E-commerce Analytics Platform

**Система сквозной аналитики для маркетплейсов Wildberries и Ozon с AI-ассистентом.**

---

## 📌 О проекте

Аналитическая платформа для управления бизнесом на маркетплейсах. Объединяет данные из API Wildberries и Ozon в единое хранилище (Baserow/PostgreSQL), автоматизирует расчёт юнит-экономики и предоставляет real-time дашборд с AI-аналитикой на русском языке.

**Стек**: Python, Streamlit, Docker, PostgreSQL, Baserow, Metabase, Qdrant, DeepSeek API, Ollama, Tailscale.

---

## 🎯 Ключевые компоненты

| Компонент | Описание |
|---|---|
| **ETL-загрузчики** | 12 загрузчиков данных из API WB и Ozon (продажи, финансы, реклама, реализация) |
| **Хранилище** | PostgreSQL через Baserow — консолидация всех данных маркетплейсов |
| **UI Dashboard** | Единый центр управления на Streamlit: статусы сервисов, финансовая сводка, управление загрузчиками |
| **AI-аналитика** | Запрос на русском → генерация и выполнение pandas-кода → интерактивный Plotly-график |
| **AI-дебаг** | Анализ кодовой базы проекта, поиск багов, устаревших endpoints, генерация исправлений |
| **Мониторинг** | Real-time статус Docker-контейнеров, Ollama-станции, Tailscale VPN |
