# 🧠 AI-Powered E-commerce Analytics Platform

**Система сквозной аналитики и автоматизации для e-commerce на маркетплейсах Wildberries и Ozon.**

> *«Не инженер, не разработчик — менеджер, который построил AI-инфраструктуру с нуля.»*

---

## 📌 О проекте

Полноценная аналитическая платформа для управления бизнесом на маркетплейсах. Объединяет данные из 5+ источников (API Wildberries, Ozon, VK Ads, Telegram Ads, 1С, банковские выписки) в единое хранилище, автоматизирует расчёт юнит-экономики и предоставляет real-time дашборд с AI-ассистентом.

**Стек**: Python, Streamlit, Docker, PostgreSQL (Baserow), Metabase, n8n, Ollama, Qdrant, Flowise.

---

## 🎯 Что сделано

| Компонент | Результат |
|---|---|
| **ETL-конвейер** | Автоматический сбор данных из API маркетплейсов и рекламных кабинетов → 9 загрузчиков, работающих по расписанию |
| **Data Lake** | Единое хранилище сырых данных в PostgreSQL (Baserow) — консолидация WB, Ozon, VK, финансы |
| **BI-аналитика** | Metabase дашборды для оперативного контроля метрик: продажи, ДРР, юнит-экономика, ROI |
| **AI-ассистент** | Запросы на русском языке → автоматическая генерация и выполнение аналитического кода → график в Metabase |
| **UI Dashboard** | Единый центр управления на Streamlit: статус компонентов, финансовая сводка, управление загрузчиками |
| **AI-инфраструктура** | Docker-контейнеры (Baserow, Metabase, Qdrant, n8n, Flowise, Ollama), Tailscale VPN |

---

## 📊 Ключевые метрики

- **Сокращение времени подготовки отчётности**: с 2 дней → 5 минут
- **Автоматизация сбора данных**: 5 API → 1 хранилище
- **Контроль рекламных бюджетов**: расчёт ДРР в реальном времени
- **AI-аналитика**: запрос на русском → готовый график
