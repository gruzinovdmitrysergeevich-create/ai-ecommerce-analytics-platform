# Инструкция для AI-агента

## Назначение

Агрегация сырых данных из Baserow в ежедневные сводки.

## Запуск

```bash
cd aggregator && python3 src/aggregator.py
```

## Структура

```
aggregator/
├── src/aggregator.py                # основной скрипт
└── configs/aggregation_rules.json   # правила агрегации
```

## Важные правила

- Не изменяй основной код без явного запроса.
- При ошибках сначала читай логи, затем диагностируй проблему.
- Не удаляй конфигурационные файлы без подтверждения.
- Ключи и пароли — только из переменных окружения.
